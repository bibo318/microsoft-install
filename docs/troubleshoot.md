# 🔧 Khắc Phục Sự Cố

## 🚀 Trước Khi Bắt Đầu

✅ **Đảm bảo** bạn đang sử dụng phiên bản mới nhất  
📸 **Chụp ảnh màn hình** lỗi để được hỗ trợ nhanh hơn  
🔄 **Khởi động lại** máy tính và thử lại  

## 💬 Liên Hệ Hỗ Trợ

Bạn gặp khó khăn? Đừng lo lắng! Chúng tôi luôn sẵn sàng giúp đỡ bạn! 🤗

### 🎮 Discord (Khuyến nghị)
Tham gia server Discord của chúng tôi để được hỗ trợ trực tiếp:
- ⚡ Phản hồi nhanh chóng
- 👥 Cộng đồng thân thiện
- 🆓 Không cần đăng ký phức tạp

### 💻 GitHub Issues
Báo cáo lỗi hoặc đề xuất tính năng mới trên GitHub:
- 🐛 Báo cáo bug chi tiết
- 💡 Đề xuất cải tiến
- 📋 Theo dõi tiến độ xử lý

**Lưu ý:** Khi liên hệ, hãy đính kèm ảnh chụp màn hình lỗi để chúng tôi có thể hỗ trợ bạn tốt nhất! 📷✨

## 🆘 Các Lỗi Thường Gặp

### 🔒 Lỗi Quyền Truy Cập
- Chạy phần mềm với quyền Administrator
- Tắt tạm thời antivirus nếu cần

### 🌐 Lỗi Kết Nối Mạng
- Kiểm tra kết nối internet
- Thử sử dụng VPN nếu bị chặn địa lý

### 💾 Lỗi Cài Đặt
- Đảm bảo đủ dung lượng ổ cứng
- Tắt các phần mềm can thiệp khác
